# Invoice Management System - Case Study

Full-stack invoice management application built with **Go** and **React**.

## Tech Stack

### Backend
- **Go** 1.23
- **Fiber**
- **GORM**
- **PostgreSQL**

### Frontend
- **React** 18
- **TypeScript**
- **Ant Design** v4
- **Vite**
